# UWash — Car Wash App

Web app to request and track car wash services in Guatemala. Built with Angular and Tailwind CSS.

## Pages

Route  Description 

| `/` | Home page |
| `/solicitar` | Form to request a car wash |
| `/paquetes` | Monthly subscription plans |
| `/mi-orden` | Track wash status by name |
| `/login` | Login page |

## Reusable Components

- `app-navbar` — navigation bar used across all pages
- `app-tipo-lavado-card` — wash type card (Básico, Medium, Premium)

## Tech Stack

- Angular 21
- Tailwind CSS
- TypeScript

## Getting Started

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
ng serve
```

Open in browser: `http://localhost:4200`
